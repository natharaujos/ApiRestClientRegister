Trabalho realizado utilizando node.js 
Foi utilizado hekoru para hospedar a api e MongoDB como banco de dados.
A aplicação pode ser acessada no link: https://trabalho-pratico-2.herokuapp.com/

Para rodar o código só é necessário o comando "node .\index.js"

Também foi feito um cliente usando python para usar a função de listar todas as tarefas
Para rodar o código do client é necessário o comando "python .\cliente.py"

Todo o trabalho foi feito com base na videoaula disponibiliza e nos códigos de exemplo do campus virtual.
